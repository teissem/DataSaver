# Data Saver

The aim of this project is to designed a data saver.

## Configuration file format

Configuration file is in JSON format :

```json
{
    "destination" : "...",
    "git" : {
        "username" : "",
        "password" : "",
        "repositories" : [
            "...", "..."
        ]
    },
    "folder" : [
        "...", "...", "..."
    ]
}
```
